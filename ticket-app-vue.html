Project Overview

Ticket-App allows users to interact with a ticketing interface for events. This project highlights my skills in frontend development using Vue.js, handling user inputs, and deploying apps on modern web hosting platforms.

Built using Vue.js, designed to handle ticket generation or ticket-related interactions for events It showcases my ability to create dynamic, interactive user experiences and manage deployment workflows.

Key Features

Fully Responsive Design

Ensured a seamless experience across all devices, from large desktop monitors to small mobile screens, using TailwindCSS techniques like Flexbox and Grid.

Performance Optimized

Achieved high performance and fast load times through image optimization, clean code, and efficient asset loading, resulting in an excellent user experience.

Smooth Scrolling & Animations

Implemented smooth scrolling for navigation and subtle on-scroll animations to make the user interface feel interactive and alive without sacrificing performance.
