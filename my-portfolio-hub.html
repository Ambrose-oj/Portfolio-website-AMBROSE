Personal Portfolio Website

A clean, professional, and responsive digital resume to showcase web development skills.

Project Overview

This project is a personal portfolio website designed for a full-stack developer. The primary goal was to create a sleek, modern, and easily navigable site that effectively showcases projects, skills, and professional background. The minimalist dark theme provides a professional aesthetic that makes visual content stand out.

The site is designed to attract new clients by clearly illustrating and explaining my services, showcasing delicious meals, and providing social proof through customer testimonials. This was a personal project to demonstrate modern front-end development skills.

Technology Stack

HTML5
CSS3
JavaScript
Netlify
Git
Figma

Key Features

Fully Responsive Design

Ensured a seamless experience across all devices, from large desktop monitors to small mobile screens, using modern CSS techniques like Flexbox and Grid.

Performance Optimized

Achieved high performance and fast load times through image optimization, clean code, and efficient asset loading, resulting in an excellent user experience.

Smooth Scrolling & Animations

Implemented smooth scrolling for navigation and subtle on-scroll animations to make the user interface feel interactive and alive without sacrificing performance.
