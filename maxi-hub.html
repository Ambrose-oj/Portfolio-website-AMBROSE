Maxi-Hub Fragrance

A special perfume website by creating an interactive, user-centric experience that helps users find scents that truly match their personality and preferences.

Project Overview

The platform combines clean aesthetics with purposeful interactivity, most notably through its Scent Finder feature, which helps users discover fragrances that match their personality and preferences. Instead of overwhelming users with choices, MAXI-HUB simplifies decision-making through a guided, intuitive flow.

MAXI-HUB Frangrance was built with scalability in mind, making it easy to expand product lines, refine the scent-matching logic, and integrate future features such as user accounts, reviews, or recommendations.

Key Features

Interactive User Experience

Interactive user experience over static browsing and a clear brand storytelling and product discovery and built with user-centric interfaces that feel intentional and refined.

Performance Optimized

Achieved high performance and fast load times through image optimization, clean code, and efficient asset loading, resulting in an excellent user experience.

Mobile First Precison, Inclusive Design

A layout that transforms seamlessly from a multi-column dashboard on desktop to a streamlined, gesture-friendly experience on mobile.
